# Idea Funder: University Project Incubation Platform

## 📌 Project Overview
Idea Funder is a full-stack platform built to help students and faculty display their academic projects. It features an admin verification system and a dedicated portal where investors can discover these projects to provide potential funding. 

## ⚙️ Key Features
* **User Authentication:** Secure login for students, faculty, and administrators.
* **Project Verification:** A dedicated workflow for admins to review and approve submitted projects before they go live.
* **Investor Dashboard:** A specialized swipe-based interface for investors to easily browse and fund approved ideas.
* **Modern UI/UX:** A fully responsive and interactive frontend experience.

## 🛠 Tech Stack
* **Frontend:** React.js
* **Backend:** Laravel (PHP)
* **Database:** MySQL
* **Security:** Laravel Auth

## 📂 Project Structure
**Backend (Laravel)**
* `app/` - Core logic and models
* `routes/` - API definitions
* `controllers/` - Business logic

**Frontend (React)**
* `components/` - Reusable UI pieces
* `pages/` - Main application views
