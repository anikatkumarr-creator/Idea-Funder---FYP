# Idea Funder: University Project Incubation Platform

## 📌 Project Overview
[cite_start]Idea Funder is a web-based platform designed for students and faculty of universities to showcase and manage projects[cite: 3]. [cite_start]It allows users to upload final-year projects or other innovative projects, get them verified by administrators, and present them to verified investors for potential funding[cite: 4]. [cite_start]The user interface includes swipe-based interactions for viewing projects and making funding decisions[cite: 6]. 

## ⚙️ Key Features
* [cite_start]**User Registration & Authentication:** Users (students and faculty) can create accounts and log in using secure authentication, while admins have elevated privileges to verify projects[cite: 12, 13].
* [cite_start]**Project Upload & Verification:** Users can upload project details, documents, and media[cite: 15]. [cite_start]Admins review and verify submitted projects before making them visible to investors[cite: 16].
* [cite_start]**Investor Access & Funding Mechanism:** Verified investors can browse through approved projects using an interactive swipe-up interface for easy navigation[cite: 18, 19]. [cite_start]Investors can fund projects directly through the platform[cite: 20].
* [cite_start]**Responsive & Interactive Design:** Built with React for a modern and smooth front-end experience, utilizing a swipe-based interactive design for easy project browsing[cite: 23, 24].

## 🛠 Tech Stack
* [cite_start]**Front-End:** React.js (Components & UI logic) [cite: 26]
